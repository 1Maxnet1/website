The *PINE H64 Model A* is a discontinued Single Board Computer by _PINE64_. It is powered by an Allwinner “H6” Quad-Core ARM Cortex A53 64-Bit Processor with a MALI T-722 GPU. For the Model B version see link:/documentation/PINE_H64_Model_B/_index[PINE H64] article.
